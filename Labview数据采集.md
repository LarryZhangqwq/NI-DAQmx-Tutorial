## 1. LabVIEW概述

### 1.1 基本信息：

## 2. LabVIEW编程环境

### 2.1 小Tips:

* 工具选板：查看 ---> 工具选板
* 导航窗口：查看 ---> 导航窗口
* 项目方式开发：文件 ---> 新建 ---> 项目 ---> 空白项目 ---> 将已打开的vi添加至新项目 ---> 弹出“项目浏览器”
* 即时帮助（节点或者vi）：右键 ---> 帮助 / 选中 ---> ctrl + h



### 2.2 一些个快捷键

* Ctrl + t : 窗口左右两栏显示
* Ctrl + u：整理框图连线
* Ctrl + shift + space：快速放置
* Ctrl + e：加载程序框图
* Ctrl + h：即时帮助
* Ctrl+b 删除所有悬空线



## 3. 编程准备知识（上）

* 控件面板中只是样式不同，功能一样
* 输入控件和显示控件的相互转换
  * 右键控件 ---> 转换为显示控件/输入控件
  * 可能前面板不会有变换，但是没法run，具体Ctrl + e 看程序框图
  * 更改会导致“断线”
* 显示和隐藏控件部件
  * 右键控件 ---> 属性 ---> 外观
* 替换前面板的对象
  * 右键控件 ---> 替换 
* 添加和缩减前面板
  *  Ctrl + 鼠标左键按住，往想要延伸的方向移动
* 对齐对象
  * 选中多个控件 ---> 工具栏中的对齐选项
* 清空前面板显示控件
  * 调出程序框图 ---> 右键输出vi ---> 创建 ---> 把新创建的vi右键转成读取 ---> 添加其他vi，连线
  * “文件”菜单 ---> vi属性 ---> 在“执行” 类别下将“调用时清空显示控件”打勾
* 为对象和背景上色
  * ”查看“菜单 ---> 工具选板 ---> 前景色 / 背景色
* 配置输入控件和显示控件
  * 选中vi ---> 右键选择“属性”
    * “启用状态”如果禁用，那么在运行时就不能随时更改输入的数值

## 4. 编程准备知识（下）

  * 单击颜色选取，右上角有个“T”
* 显示所有隐藏控件（只能在自定义控件和全局变量中使用）
  * 工具栏“编辑” ---> 显示隐藏的输入控件和显示控件
* 隐藏控件
  * 右键控件 ---> 高级 ---> 隐藏xx控件
* 显示前面板的对象
  *  切换到程序框图 ---> 右键控件 ---> 显示xx控件
* 显示控件的滚动条（当字符串控件中字数太多显示不过来）
  * 右键控件 ---> 属性 ---> 外观 ---> 显示垂直滚动条
* 分布对象（前面板）
  * 工具栏“分布对象” 
* 组合和锁定对象
  * 多选控件 ---> 工具栏中“重新排序” ---> 组 
  * 锁定：”重新排序“ ---> 锁定
* 重新排列对象（前面板的控件出现堆叠，在运行中产生问题）

* 缩放前面板对象
  * 右键控件 ---> 根据窗格缩放对象
* 设置控件的快捷键
  * 右键控件 ---> ”快捷键“
* 设置Tab键顺序
  * 可以依赖键盘在run时选中不同的输入控件
  * 菜单“编辑” ---> “设置Tab键顺序” ---> 按顺序点击控件
* 显示/隐藏标签以及数字显示
  * 右键控件 ---> 显示项 ---> 数字显示
* 导入窗格背景
  * 右键右侧滚动条 ---> 属性 ---> 背景
* Windows平台的图像导入

## 6. 程序框图的连线

* 连线的外观和结构
  * 数据类型查看
    * ctrl + h单击连线可查看连线数据类型
    * 右键常量/数值控件 ---> 表示法
    * 通过连线的颜色、粗细、条数判断
  * 判断断线原因
    * 运行图表会断掉，单击运行会列出错误
  * 手动连线
    * 光标移动到控件两侧，会出现连线卷的样式，单击即可进行连线
  * 自动连线
    * 移动两个控件使其靠近，靠近到一定程度时会出现一个连线，单击鼠标左键即可完成自动连线（注意两个控件要连线的地方要对齐）
  *  关闭自动连线
    * 菜单栏“工具” ---> 选项 --->  程序框图 ---> 向下翻 ---> 连线 ---> “启用自动连线”
  * 连线的路径选择
    * 连线过程中在想设置直角的地方单击鼠标即可人为创造一个直角转折，人为规划路径
  * 选择连线
    * 单击鼠标左键 --->  选中一段连线
    * 双击鼠标左键 ---> 选中所有人为规划的连线（无分支）
    * 三次鼠标左键 ---> 选中本连线的所有分支
  * 在连线上加标签
    * 右键选中的连线 ---> 显示项 ---> 标签
  * 纠正断线
    * ctrl+b 删除所有悬空线

## 15. 数值型数据——浮点数

* 数值的数据类型
  * 浮点数
    * 单精度（SGL）- 32位IEEE单精度格式
    * 双精度（DBL）- 64位IEEE双精度格式
    * 扩展精度（EXT）- LabVIEW将其保存位独立于平台的128位格式
  * 定点数
  * 整数型
  * 不带符号的整形数
  * 复数
* 浮点数
  * NaN：非法数字（比如对负数开平方）
  * 单位：右键选中的控件 ---> 单位标签
  * 寻找当前数据类型支持的单位：
    * 右键单位的地方 ---> 穿件单位字符串
  * 更改数字显示格式：
    * 右键控件 ---> 显示格式

## 16. 数值型数据——定点数

* 定点数可以表示用户指定范围和精度内的有理数，大小可以从第一位到64位，可以配置为带符号或不带符号的。定点数据类型是一种以二进制数（又称“位”）表示一组有理数的数值数据类型。定点数的整数和小数部分都是定长不可变的。
* 定点数表示有理数
  * 定点数具有固定个数的整数位和分数位。整数位和分数位分别在二进制点点左右两侧。由于定点数可制定起确切的位数，故二进制小数点的位置是固定的
  * 在“表示法”选项中在第一行的最后一个
* 配置定点数
  * 右键“属性” ---> 数据类型
  * Delta是步长，定点数只能表示为 min + n * delta 的数
  * 输入的不是min + n * delta 的数时，会近似到最近的合法数
* 定点数的范围
* 显示和隐藏定点数溢出状态
  * 右键“属性” ---> 数据类型 ---> “包括溢出状态”
  * 如果选中，则在前面板中的控件左侧会出现一个LED灯
  * 溢出时LED会亮
* 支持定点数据类型的函数和VI：
  * 数值函数、比较函数、转换函数、数据操作函数、LabVIEW模式函数、字符串/数值转换函数

## 17. 数值型数据——整数型

* “表示法”界面的第二行
  * 整型数分为单子节整型（ I8、-128 - 127 ）、双字节整型（ I16、-32768 - 32767 ）、长整型（ I32、-2147483648 - 2147483647 ）、64位整型（I64）
* 当从浮点数转换为整形数时会**五舍六入**

## 18. 数值型数据——无符号整型数

* 只能表示非负整型数
  * 单子节整型（U8）max: 255
  * 双字节整型数（U16）max: 65535
  * 长整型（U32）max: 4294967295
  * 64位整型（U64）
* “表示法”界面的第三行

## 19. 数值型数据——复数

* 三种不同的类型
  * 单精度浮点复数（CSG）
  * 双精度浮点复数（CDB）
  * 扩展精度浮点复数（CXT）
  * 他们与对应类型的浮点数相同，但是有实部和虚部
* "表示法"界面的第四行
* 通过搜索“复数”在函数中寻找复数相关的函数操作以及控件
